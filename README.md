# 图书管理系统简介 java925

## 项目概述

本系统是基于Spring Boot和Vue3的前后端分离图书管理系统，旨在提供一套完善的图书管理解决方案。

## 技术栈

- 后端：Spring Boot
- 前端：Vue3 + Element-UI

## 运行环境

- JDK 1.8
- Mysql 5.5
- IDEA
- Node.js
- Maven

## 系统角色

系统分为管理员和普通用户两种角色。

### 管理员

- **图书信息管理**：支持搜索、新增、编辑修改图书，以及借阅图书。
- **图书类型管理**：支持搜索、添加、编辑和删除图书类型。
- **借阅信息管理**：支持新增、修改和归还图书。
- **用户管理**：支持搜索、新增、修改、查询用户。
- **密码更改**：可以更改密码。

### 用户

- **图书管理**：支持搜索和借阅图书。
- **借阅图书管理**：支持归还图书。
- **其他管理**：支持修改密码。

## 功能模块

系统的核心功能模块包括但不限于：

- 图书信息模块
- 图书类型模块
- 借阅信息模块
- 用户管理模块
- 密码管理模块

## 目录结构

```
├── backend    // 后端代码
├── frontend   // 前端代码
└── sql        // 数据库脚本文件
```

## 核心亮点

- **技术先进性**：基于Spring Boot和Vue3的最新技术，确保系统性能与可维护性。
- **角色分明**：管理员和用户角色清晰，各司其职，易于管理。
- **功能完善**：包含图书及用户管理等多个功能模块，满足日常图书管理需求。
